Jiconstruct Construction Company Documentation

Welcome to the official documentation for Jiconstruct, a construction company website showcasing location, contact, projects, news, and more.

Project Overview

This project is a web application built for Jiconstruct. The frontend is developed with React, and the backend API is powered by Express.

Setup Instructions

Installation

git clone <repository-url>
cd project-directory
npm install
